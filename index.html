What is Protowrite?

A plain text to HTML converter

Protowrite is a plain text to HTML converter. It allows authors to write in plain text with virtually zero markup, and then convert their documents to HTML. The output is unambiguous and suitable for professional publication. That might sound impossible, but it's not, and Protowrite is here to prove it.

Protowrite was created to make writing for online publication better. It caters to humans, not machines.

A plain text syntax

Protowrite is a plain text syntax based on how people naturally write. It uses newlines, tabs, and spaces to differentiate elements like headings, paragraphs, lists, and block quotes. In fact, a document conformant with the Protowrite syntax contains virtually no markup. This makes source documents exceptionally easy to read, write, and edit.

And by the way, the Protowrite syntax fully supports images, audio, video, and embeded content.
