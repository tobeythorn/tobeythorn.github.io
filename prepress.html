What is Prepress?

A plain text to HTML converter

Prepress is a plain text to HTML converter. It let's writers write in plain text with virtually no markup, and then convert their documents to HTML. The output is suitable for professional publication.

Prepress was created so that people can just write. It caters to humans, not machines. You can read more about the philosophy here.

Prepress is available on the following platforms:

  1. Prepress.py
  2. Jekyll (coming soon)
  3. Hugo (coming soon)

A plain text format

Prepress is a plain text format based on how people naturally write. It uses newlines, tabs, and spaces to differentiate elements like headings, paragraphs, lists, and block quotes. This makes source documents exceptionally easy to read, write, and edit.

And by the way, Prepress fully supports images, audio, video, and embeded content.
